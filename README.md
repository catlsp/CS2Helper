# CS2 Helper App

**CS2 Helper** — это Android приложение-помощник для игроков в Counter-Strike 2. Здесь собраны раскидки гранат, названия позиций на картах, коды прицелов и секретная мини-игра, чтобы скоротать время поиска матча.

##  Основной функционал

*   *Раскидки гранат:*
    *   База данных смоков, флешек и молотовых для основных карт.
    *   Фильтрация по типу гранаты и стороне (T/CT).
    *   Пошаговая инструкция с фото (позиция, прицел, результат).
*   Позиции на картах:
    *   Схемы карт с названиями позиций.
*   Прицелы:
    *   Коды прицелов про-игроков (s1mple, NiKo, m0NESY и др.).
    *   Копирование кода в буфер обмена одним нажатием.
*   Гайды:
    *   Ссылки на тренировочные карты в Steam Workshop.
*   Секретная мини-игра
